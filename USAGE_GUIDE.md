# 🎵 WWM Music Auto-Player — Полный Гайд

## Быстрый Старт

1. Запустить `START.bat` (или `python main.py`)
2. Нажать **Import MIDI** → выбрать `.mid` файл
3. Выбрать трек(и) и опции конвертации
4. Нажать **Play** → переключиться в игру за 3 секунды
5. Убедиться, что инструмент открыт в игре!

---

## 📋 Все Опции Конвертации

При импорте MIDI открывается окно с множеством опций. Вот что каждая делает и когда её использовать.

### 🎹 Select Target Instrument
**Что делает:** Выбирает диапазон нот инструмента (каждый инструмент в игре имеет свой диапазон).

| Инструмент | Диапазон | Когда использовать |
|---|---|---|
| Default (Piano) | C3-B5 (48-83) | Стандартный вариант для большинства мелодий |
| Guqin (古琴) | C2-B4 (36-71) | Низкий диапазон, подходит для басовых мелодий |
| Pipa (琵琶) | A2-E5 (45-80) | Средний диапазон, подходит для плавных мелодий |
| Erhu (二胡) | C4-B5 (60-83) | Высокий диапазон, для мелодий в верхнем регистре |
| Fangxiang (方响) | C4-B6 (60-95) | Очень высокий, для колокольчиков и перезвонов |
| Konghou (箜篌) | C3-B5 (48-83) | Аналог Default, для арфового звука |

---

### ✅ Quantize (Align to 1/16 Grid)
**По умолчанию:** ВКЛ

**Что делает:** Выравнивает все ноты по сетке 1/16, убирая микро-неточности в тайминге.

| Ситуация | Включать? |
|---|---|
| Обычная мелодия / поп / рок | ✅ Да |
| Рубато / свободный ритм / классика | ❌ Нет |
| YouTube-транскрипция (много шума) | ✅ Да |

---

### 🎶 Play Chords (Fast Arpeggio)
**По умолчанию:** ВЫКЛ

**Что делает:** Вместо одной ноты за раз, играет все одновременные ноты быстрой последовательностью (арпеджио ~30мс на ноту).

| Ситуация | Включать? |
|---|---|
| Одна мелодия без аккордов | ❌ Нет |
| Песня с аккордами, хочется «полноты» | ✅ Да |
| Сложные аранжировки (4+ треков) | ❌ Нет (будет каша) |

---

### 🔇 Ignore Low Bass (Fix 'Messy' Audio)
**По умолчанию:** ВЫКЛ

**Что делает:** Удаляет ноты ниже минимальной границы инструмента. Полезно если бас создаёт «грязь».

| Ситуация | Включать? |
|---|---|
| Несколько треков и бас мешает | ✅ Да |
| Один трек мелодии | ❌ Нет |
| Мелодия в низком регистре | ❌ Нет (потеряешь ноты!) |

---

### ✂️ Simplify Chords (Keep Melody+Bass Only)
**По умолчанию:** ВЫКЛ

**Что делает:** При включённых аккордах, оставляет только самую высокую (мелодия) и самую низкую (бас) ноту, убирая середину.

| Ситуация | Включать? |
|---|---|
| Аккорды звучат «грязно» / слишком плотно | ✅ Да (вместе с Chords) |
| Нет аккордов или мелодия монофоническая | ❌ Нет (нет эффекта) |

---

### 🎵 Smart Melody (Recognizable Melody Extraction)
**По умолчанию:** ВЫКЛ

**Что делает:** Умный алгоритм извлечения мелодии. Из всех одновременных нот выбирает самую «мелодичную» по нескольким критериям:
- Ноты в вокальном диапазоне (C4-C6) получают бонус
- Ноты из Lead Track получают максимальный приоритет
- Плавное движение (маленькие интервалы) предпочтительнее скачков
- Высокие ноты слегка предпочтительнее (мелодия обычно сверху)

Также включает **Interval-Preserving Snap** — при снэппинге хроматических нот на диатонические, сохраняет направление мелодии (если оригинал шёл вверх, снэпнутая версия тоже идёт вверх).

| Ситуация | Включать? |
|---|---|
| Несколько треков, нужна чистая мелодия | ✅ Да |
| YouTube-транскрипция (много мусора) | ✅ Да |
| Один трек уже содержит только мелодию | ❌ Нет (замедлит без пользы) |
| Хочешь услышать аккорды и полифонию | ❌ Нет |

---

### 🥁 Rhythm Filter (Remove noise, keep beat pattern)
**По умолчанию:** ВЫКЛ

**Что делает:** Анализирует ритмический паттерн песни и удаляет ноты, которые не попадают в основной ритм. Полезно для шумных транскрипций.

Алгоритм:
1. Делит музыку на такты
2. Находит повторяющийся ритмический «отпечаток»
3. Удаляет ноты, не совпадающие с паттерном (±1 слот допуска)

| Ситуация | Включать? |
|---|---|
| YouTube/аудио транскрипция со множеством шума | ✅ Да |
| Чистый MIDI-файл | ❌ Нет |
| Песня с нерегулярным ритмом | ❌ Нет (удалит нужные ноты) |

---

### 🎼 Smart Multi-Track (Per-track transposition) ⭐ НОВОЕ
**По умолчанию:** ВЫКЛ

**Что делает:** Кардинально улучшает звучание при конвертации **нескольких треков одновременно**. Вместо одного глобального сдвига для всех треков, каждый трек получает свой оптимальный октавный сдвиг.

Включает три механизма:
1. **Per-Track Shift** — каждый трек транспонируется так, чтобы его медиана попала в центр диапазона
2. **Register Allocation** — треки распределяются по регистрам (low/mid/high), чтобы не конфликтовать
3. **Soft Clamp** — ноты вне диапазона мягко обрезаются вместо октавного заворачивания (если нота слишком далеко — пропускается)

| Ситуация | Включать? |
|---|---|
| **2+ треков**, мелодия звучит как каша | ✅ **ДА, это для тебя!** |
| 1 трек | ❌ Нет (нет эффекта, работает только при 2+ треках) |
| Хочешь максимально точное воспроизведение всех нот | ❌ Нет (пропускает ноты вне диапазона) |

---

### 🎚️ Manual Octave Shift
**По умолчанию:** 0

**Что делает:** Ручной сдвиг всей мелодии на N октав вверх (+) или вниз (-).

| Значение | Эффект |
|---|---|
| 0 | Автоматический центр (рекомендуется) |
| +1, +2 | Мелодия выше (для басовых MIDI-файлов) |
| -1, -2 | Мелодия ниже (для высоких MIDI-файлов) |

---

### 🎯 Lead Track Selection
**Появляется при:** Выборе 2+ треков

**Что делает:** Указывает, какой трек является главной мелодией. Ноты этого трека получают приоритет при конфликтах (совпадении по времени).

| Вариант | Когда |
|---|---|
| Mix All Equally | Не уверен, какой трек главный |
| Конкретный трек | Знаешь точно, где мелодия |

## 🎯 Рецепты: Комбинации Опций

### «Хочу просто мелодию, чисто и узнаваемо»
- ✅ Quantize
- ✅ Smart Melody
- ❌ Всё остальное

### «Несколько треков, хочу полное звучание»
- ✅ Quantize
- ✅ **Smart Multi-Track** ⭐
- ✅ Выбрать Lead Track
- По желанию: Smart Melody (для ещё более чистого звука)

### «YouTube-транскрипция, много мусора»
- ✅ Quantize
- ✅ Smart Melody
- ✅ Rhythm Filter
- ✅ Ignore Low Bass

### «Один трек с аккордами, хочу звучание побогаче»
- ✅ Quantize
- ✅ Play Chords
- ✅ Simplify Chords (если аккорды слишком плотные)

### «Классика / свободный ритм»
- ❌ Quantize
- ✅ Smart Melody
- ❌ Rhythm Filter
