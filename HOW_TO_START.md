# 🚀 Как запустить приложение / How to Start

## ✅ Способ 1: Двойной клик (САМЫЙ ПРОСТОЙ)

1. Найдите файл **`START.bat`** в папке проекта
2. **Два раза кликните** на него
3. Приложение запустится автоматически!

---

## ✅ Способ 2: Через командную строку

### Вариант A: PowerShell
1. Нажмите **Win + X** → выберите **"Windows PowerShell"**
2. Перейдите в папку проекта:
   ```powershell
   cd path\to\wwm-music
   python main.py
   ```

### Вариант B: CMD
1. Нажмите **Win + R** → введите **`cmd`** → Enter
2. Перейдите в папку проекта:
   ```cmd
   cd path\to\wwm-music
   python main.py
   ```

---

## ❌ Если видите ошибку "python не найден"

### Решение: Установите Python

1. Скачайте Python с [python.org](https://www.python.org/downloads/)
2. При установке **ОБЯЗАТЕЛЬНО** поставьте галочку **"Add Python to PATH"**
3. Установите
4. Перезапустите командную строку
5. Попробуйте снова: `python main.py`

### Проверка что Python установлен:
```cmd
python --version
```
Должно показать: `Python 3.x.x`

---

## ❌ Если видите ошибку "No module named ..."

### Решение: Установите зависимости

**Базовая установка** (MIDI + воспроизведение):
```cmd
pip install -r requirements.txt
```

**Полная установка** (+ YouTube + AI транскрипция):
```cmd
pip install -r requirements-full.txt
```

Подождите пока установятся все библиотеки, потом запустите приложение снова.

---

## ❌ Если приложение закрывается сразу

Возможная проблема — ошибка импорта или синтаксиса.

### Проверьте ошибку:
Откройте командную строку, перейдите в папку проекта и запустите:
```cmd
python main.py
```

Просмотрите текст ошибки и создайте Issue на GitHub с описанием!

---

## 🎯 Быстрая диагностика

Если приложение не запускается, последовательно проверьте:

1. ✅ Python установлен? → `python --version`
2. ✅ Зависимости установлены? → `pip list | findstr pynput`
3. ✅ В правильной папке? → `dir main.py` (должен показать файл)
4. ✅ Запускается? → `python main.py`

Если всё это "ОК", но приложение не работает — создайте Issue с точным текстом ошибки!

## ✅ Что вы должны увидеть

- Окно с названием **"WWM Music Auto-Player"**
